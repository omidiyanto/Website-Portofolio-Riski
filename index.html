IBM Academy : Hybrid Cloud & Red Hat

Infinite Learning partners with IBM to provide the Red Hat System Administrator independent study program. This program builds the foundation for essential Linux system administration tasks, including installation, network connectivity, physical storage management, and basic security administration. Students will also learn more advanced topics like storage management using LVM, SELinux management, and automated installations.
